### 풀이 코드

```java
package com.ct.practice.baekjoon;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Stack;

public class B10799 {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String input = br.readLine();
        Stack<Character> stack = new Stack<>();
        int result = 0;

        for (int i = 0; i < input.length(); i++) {
            char target = input.charAt(i);

            if (target == '(') {
                stack.add(target);
            } else { // )

                if (input.charAt(i - 1) == '(') { // 레이저라면
                    stack.pop();
                    result += stack.size();
                } else {
                    stack.pop();
                    result += 1;
                }
            }
        }
        System.out.println(result);

    }

}

```

- `Stack` Character가 담긴다..
- `stack.size()` 를 이용하는 방법을 문제를 통해 파악할 수 있을까? 아마 못했을 것 같음.
- 즉 코드가 말하고자 하는바를 아직 정확히 파악하고 있진 못하다



### 다른 풀이

```java
package com.ct.practice.baekjoon;

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Stack;

public class B10799Integer {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String input = br.readLine();
        Stack<Integer> stack = new Stack<>();
        int result = 0;

        for (int i = 0; i < input.length(); i++) {
            char target = input.charAt(i);

            if (target == '(') {
                stack.add(i);
            } else { // ')' 일경우

                if (stack.peek() == i - 1) { // 레이저라면
                    stack.pop();
                    result += stack.size();
                } else {
                    stack.pop();
                    result += 1;
                }
            }
        }
        System.out.println(result);

    }

}
```
