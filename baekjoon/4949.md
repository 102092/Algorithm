[링크](https://www.acmicpc.net/problem/4949)

```java
package app;

import java.util.Scanner;
import java.util.Stack;


public class App {
    public static void main(String[] args) throws Exception {
        Scanner sc = new Scanner(System.in);
        

        while(true){
            
            String input = sc.nextLine();
            char temp;
            Stack<Character> stk = new Stack<Character>();
            boolean isBal = true;
            if (input.length() == 1) //이부분 포인트
                break;
            

            for(int i =0; i<input.length(); i++){
                temp = input.charAt(i);

                if(temp =='.') 
                    break;

                if(temp == '(' || temp =='['){
                    stk.push(temp);
                }
                else if(temp == ')'){
                    if(stk.isEmpty() || stk.pop() != '('){
                        isBal = false;
                        break;
                    }
                }
                else if(temp == ']'){
                    if(stk.isEmpty() || stk.pop() != '['){
                        isBal = false;
                        break;
                    }
                }


            }
            if(!stk.isEmpty()){
                isBal = false;
            }
            if(isBal){
                System.out.println("yes");
            }else{
                System.out.println("no");
            }
            
        }
    }
}
```



- 마지막 `.` 까지 true/false 처리를 해서인지, 혹은 계속 중간 `break` 문 없이 while을 돌려서 그런지 `런타임에러` 가 났음.

- 그래서 위의 부분 즉 `input.length()` 가 하나가 들어오면 다시 말해 `.` 이 들어오면 break를 걸어줬더니 `런타임에러`가 나지 않았음.

  약간의 트릭같은 스킬이라 생각됨.
